---------------------------------------------------------------------------------------

Ass 4 is a server-based version of Ass 1.
It is limited to one connection at a time (using TCP).

Nearly identical to Ass1, except that the server handles all function calls and only communicates with the client to pass information to it or ask for further input.


The code handles basic exceptions.  len checks, isalpha checks, and checks for validity of text files are used.
